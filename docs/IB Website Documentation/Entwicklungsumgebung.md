## Wichtige Befehle
Starten der Umgebung
```bash
ddev start
```
Installiert alle Abhängigkeiten und baut die Pakete
```bash
ddev yarn install
ddev yarn run dev
```
Kontinuierliche Überwachung und stetes Bauen der Pakete
```bash
ddev yarn run watch
```
Bauen der Pakete für das Deployment
```bash
ddev yarn build
```
