=========================================
Apache Solr for TYPO3 - Console tools 3.0
=========================================

We are happy to release EXT:solrconsole 3.0.0.
The focus of this release was the compatibility with EXT:solr v10.0 and the site handling support.

New in this release
===================

[TASK] Make compatible with EXT:solr 10
---------------------------------------

Makes solrconsole compatible with EXT:solr 10 by removing commands in none legacy mode
